# 다나와 스펙 크롤링 및 경쟁사 비교 시스템 구현 참조 자료

## 1. 다나와 크롤링 기본 가이드 (파이썬)
다나와는 봇 방지(Anti-bot) 시스템이 적용되어 있을 확률이 높으므로 단순 `requests`나 `BeautifulSoup` 대신 브라우저 렌더링을 활용하는 `Selenium` 또는 `Playwright`를 권장합니다.

### 1.1 기술 스택 추천
- **Playwright (Python)**: Selenium보다 빠르고 headless 모드에서 최신 웹 표준을 잘 지원함. 대기(Wait) 로직이 내장되어 있어 크롤링 안정성이 높음.
- **BeautifulSoup**: Playwright로 렌더링된 HTML을 파싱할 때 결합해서 쓰기 좋음.

### 1.2 검색 및 정보 추출 흐름
1. `https://search.danawa.com/dsearch.php?query=[모델명]` URL로 접속
2. 첫 번째 검색 결과(보통 정확히 일치하는 모델)의 상세 페이지 링크 클릭
3. 상세 페이지 하단의 '상품 상세 스펙' 테이블 안의 항목 파싱
4. 텍스트 정규화 (예: '화면크기: 163cm(65인치)' -> '65')

## 2. 주요 스펙 vs 등급 스펙 분류 로직 (가상 예시: TV)
사용자가 직접 로직을 세팅할 수 있어야 하지만, 기본적으로 아래와 같이 분류합니다.
- **필수 스펙 (고정값 필터링용)**:
  - 품목: LED TV, OLED TV 등
  - 크기: 65인치, 75인치 등
  - 해상도: 4K UHD, 8K 등
- **등급 스펙 (채점/가중치용)**:
  - 주사율 (60Hz, 120Hz, 144Hz) -> 최대 점수 30점
  - 소비전력 등급 (1등급~5등급) -> 최대 점수 10점
  - 부가 기능 (스마트, AI업스케일링, 돌비비전 등 유무) -> 기능당 5점

## 3. 경쟁사 모델 추출 방법
1. 삼성 모델명으로 검색하여 해당 모델이 속한 '다나와 카테고리'에 진입.
2. 상세 검색 필터를 활용하여 **[필수 스펙]**에 해당하는 옵션(예: 65인치, 4K, OLED)을 체크하고 제조사를 경쟁사(LG, TCL 등)로 지정.
3. 인기 상품순 1~10위의 상품 링크를 수집 (상위 10개).

## 4. 점수 유사도 기반 리스팅 로직 (KNN 또는 단순 오차 비교)
- 타겟 모델 점수: $S_t$ (100점 만점)
- 경쟁사 모델 점수: $S_c$ (100점 만점)
- 점수 차이 $\Delta S = |S_t - S_c|$ 가 가장 적은 순서대로 정렬. 추가적으로 가격 ($P$) 을 함께 고려하여 가성비를 노출하면 더욱 좋음.
